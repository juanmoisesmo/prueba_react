## Pasos para empezar

1: Abre el proyecto en tu editor <br>
2: Localizate en tu proyecto redux-crud <br>
3: Pon el comando npm install para instalar las dependencias. <br>

### `npm start`

Ya instaladas las dependencias escribe:<br> npm start
